# Contributing to VotePeace

Thank you for your interest in contributing! We welcome contributions to make this project better.

## How to Contribute

1.  **Fork the Repository**: Click the "Fork" button on the top right of the GitHub repository.
2.  **Clone your Fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/VotePeace-Backend.git
    ```
3.  **Create a Branch**:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Make Changes**: Implement your feature or fix.
5.  **Commit Changes**:
    ```bash
    git commit -m "feat: add amazing feature"
    ```
    *Please follow [Conventional Commits](https://www.conventionalcommits.org/).*
6.  **Push to Branch**:
    ```bash
    git push origin feature/amazing-feature
    ```
7.  **Open a Pull Request**: Submit your PR on the main repository.

## Guidelines

-   Please verify your code compiles and runs (`go run main.go`).
-   Run `go fmt ./...` to format your code.
-   Keep pull requests focused on a single feature or fix.

## Reporting Issues

If you find a bug, please open an issue describing:
-   Steps to reproduce.
-   Expected behavior.
-   Actual behavior.
