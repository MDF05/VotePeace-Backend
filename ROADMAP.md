# Roadmap

We are constantly improving VotePeace. Here is the plan for future development.

## Phase 1: Core Foundation (Current)
- [x] User Authentication
- [x] Campaign CRUD
- [x] Voting Logic
- [x] Basic Analytics

## Phase 2: Enhanced Features
- [ ] **Email Verification**: Integration with SMTP for account creation.
- [ ] **Candidate Profiles**: Rich text description and video uploads.
- [ ] **Real-time Updates**: WebSockets for live vote counting on the dashboard.

## Phase 3: Scaling
- [ ] **Docker Support**: Full containerization.
- [ ] **PostgreSQL**: Migration from SQLite to Postgres for concurrency.
- [ ] **Redis**: Caching for high-traffic vote counting.
